<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
```sh
npm install npm@latest -g
```
* yarn
```sh
npm install yarn -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/RobertoNobre/weather-pwa.git
```
3. Install NPM packages
```sh
yarn
```
4. Run
```JS
yarn start
```
